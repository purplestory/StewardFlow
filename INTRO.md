# StewardFlow 소개

## 교회 자원을 더 효율적으로 관리하고 공유하세요

StewardFlow는 교회 내 물품, 공간, 차량을 통합 관리하고 효율적으로 공유할 수 있는 웹 플랫폼입니다. 각 부서가 보유한 자원을 한눈에 파악하고, 필요할 때 쉽게 대여할 수 있도록 도와드립니다.

---

## 왜 StewardFlow를 사용해야 할까요?

### 📦 자원 가시화
**문제**: "어디에 어떤 물품이 있는지 모르겠어요"
- 각 부서에 어떤 물품이 있는지 한눈에 확인
- 검색 기능으로 원하는 물품을 빠르게 찾기
- 상태별 필터링으로 사용 가능한 물품만 확인

**해결**: 모든 자원을 한 곳에서 확인하고 관리

### 🔄 효율적인 공유
**문제**: "다른 부서에 있는 물품을 빌리고 싶은데 누구에게 연락해야 할지 모르겠어요"
- 온라인으로 간편하게 대여 신청
- 예약 캘린더로 사용 가능한 날짜 확인
- 승인 프로세스로 체계적인 관리

**해결**: 복잡한 연락 없이 온라인으로 간편하게 대여

### 📱 모바일 최적화
**문제**: "현장에서 물품을 등록하거나 확인하기 어려워요"
- 스마트폰에서 바로 접근 가능
- 사진 촬영 후 즉시 등록
- 언제 어디서나 자원 정보 확인

**해결**: 현장에서도 쉽게 사용할 수 있는 모바일 친화적 인터페이스

### ✅ 체계적인 관리
**문제**: "누가 무엇을 빌렸는지, 언제 반납해야 하는지 관리가 어려워요"
- 예약 현황을 한눈에 확인
- 반납 알림으로 놓치지 않는 관리
- 감사 로그로 모든 활동 기록

**해결**: 체계적이고 투명한 자원 관리 시스템

## 주요 기능

### 1. 물품 관리
- **등록**: 부서별 보유 물품을 사진과 함께 등록
- **조회**: 전체 물품을 검색하고 필터링하여 확인
- **대여**: 필요한 물품을 온라인으로 신청하고 승인받기
- **상태 관리**: 대여 가능, 대여 중, 수리 중, 분실, 불용품 상태 관리

### 2. 공간 관리
- **공간 등록**: 예배실, 교육실 등 공간 정보 등록
- **예약 관리**: 날짜와 시간을 선택하여 공간 예약
- **사용 현황**: 예약된 공간과 사용 가능한 공간 확인

### 3. 차량 관리
- **차량 등록**: 교회 차량 정보 등록
- **사용 신청**: 차량 사용 일정 신청
- **주행거리 추적**: 대여 시 초기 주행거리 기록

### 4. 예약 시스템
- **캘린더 뷰**: 한눈에 보는 예약 현황
- **승인 프로세스**: 역할 기반 승인 시스템
- **알림**: 예약 신청, 승인, 반납 알림

### 5. 불용품 양도
- **양도 요청**: 다른 부서의 불용품을 우리 부서로 양도 요청
- **승인 관리**: 관리자가 양도 요청을 승인/거절
- **효율적 활용**: 사용하지 않는 물품을 다른 부서에서 활용

## 사용자별 역할

### 👤 일반 사용자
**할 수 있는 것:**
- 물품, 공간, 차량 조회 및 검색
- 대여 신청
- 불용품 양도 요청
- 프로필 정보 수정
- 부서 변경 요청

**접근 불가:**
- 관리 페이지
- 예약 승인/반려
- 물품 등록/수정

### 👨‍💼 부서 관리자
**할 수 있는 것:**
- 일반 사용자의 모든 기능
- 자신의 부서 물품 등록/수정
- 자신의 부서 예약 승인/반려
- 같은 부서 사용자 역할 변경
- 부서 변경 요청 승인

**접근 불가:**
- 다른 부서 물품 관리
- 기관 설정
- 사용자 부서 변경

### 👑 최고 관리자
**할 수 있는 것:**
- 모든 기능 접근
- 기관 설정
- 부서 관리
- 모든 사용자 관리
- 역할 및 부서 변경
- 승인 정책 설정

## 시작하기

### 1단계: 가입하기
- **초대 코드로 가입**: 관리자로부터 받은 초대 코드 입력
- **카카오톡 로그인**: 간편하게 카카오톡 계정으로 가입 (관리자 승인 필요)

### 2단계: 프로필 설정
- 전화번호 입력
- 소속 부서 확인
- 프로필 정보 확인

### 3단계: 자원 둘러보기
- 물품, 공간, 차량 목록 확인
- 검색 기능 사용해보기
- 상세 정보 확인

### 4단계: 첫 대여 신청
- 원하는 물품/공간/차량 선택
- 예약 캘린더에서 날짜 선택
- 사용 목적 입력 후 신청

## 사용 시나리오

### 시나리오 1: 주일 예배 음향 지원
**상황**: 주일 예배에 음향 장비가 필요합니다.

1. StewardFlow 접속
2. "물품" 메뉴에서 "음향" 카테고리 선택
3. 사용 가능한 음향 장비 확인
4. 원하는 장비 선택 후 "대여 신청"
5. 예약 캘린더에서 주일 날짜 선택
6. 사용 목적 입력: "주일 예배 음향 지원"
7. 신청 완료 → 관리자 승인 대기
8. 승인 알림 수신 후 사용

### 시나리오 2: 교육실 예약
**상황**: 다음 주 토요일에 교육실을 사용하고 싶습니다.

1. "공간" 메뉴 접속
2. 사용 가능한 교육실 확인
3. 원하는 교육실 선택
4. 예약 캘린더에서 다음 주 토요일 선택
5. 시작 시간과 종료 시간 입력
6. 사용 목적 입력 후 신청
7. 승인 후 사용

### 시나리오 3: 불용품 활용
**상황**: 다른 부서에 사용하지 않는 물품이 있어 우리 부서에서 활용하고 싶습니다.

1. 해당 물품 상세보기 접속
2. "불용품 양도 요청" 섹션 확인
3. 요청 사유 입력
4. "이동 요청" 클릭
5. 관리자 승인 대기
6. 승인 후 물품 소유권 이전

## 자주 묻는 질문 (FAQ)

### Q1. 초대 코드 없이 가입할 수 있나요?
**A**: 네, 카카오톡 로그인으로 가입할 수 있습니다. 다만 관리자 승인이 필요하며, 승인 전까지는 메인 페이지만 이용할 수 있습니다.

### Q2. 예약은 누가 승인하나요?
**A**: 기관의 승인 정책에 따라 다릅니다. 부서 관리자 승인이 필요한 경우 부서 관리자가, 관리자 승인이 필요한 경우 관리자가 승인합니다.

### Q3. 물품을 등록하려면 어떻게 해야 하나요?
**A**: 부서 관리자 또는 최고 관리자 권한이 필요합니다. 자원 관리 메뉴에서 "물품 등록" 버튼을 클릭하여 등록할 수 있습니다.

### Q4. 예약을 취소할 수 있나요?
**A**: 예약 상태에 따라 다릅니다. 승인 대기 중인 경우 취소할 수 있으며, 이미 승인된 경우 관리자에게 문의하시기 바랍니다.

### Q5. 다른 부서의 물품도 빌릴 수 있나요?
**A**: 네, 물품 소유 범위가 "기관 공용"이거나 해당 부서가 대여를 허용한 경우 빌릴 수 있습니다.

### Q6. 모바일에서도 사용할 수 있나요?
**A**: 네, 스마트폰 브라우저에서도 완벽하게 작동합니다. 모바일에 최적화된 인터페이스로 제공됩니다.
