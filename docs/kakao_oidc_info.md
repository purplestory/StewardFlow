# 카카오 OpenID Connect 설정 가이드

## OpenID Connect란?

OpenID Connect (OIDC)는 OAuth 2.0 위에 구축된 인증 레이어로, 사용자 인증 정보가 담긴 **ID 토큰**을 추가로 발급받을 수 있습니다.

## Supabase에서 OpenID Connect가 필요한가?

### 기본 OAuth 2.0만으로도 동작합니다
- Supabase는 기본 OAuth 2.0만으로도 카카오 로그인을 처리할 수 있습니다
- 사용자 인증은 정상적으로 작동합니다

### OpenID Connect를 활성화하면 좋은 점
1. **더 많은 사용자 정보**: 이메일, 프로필 사진, 닉네임 등을 안정적으로 받을 수 있습니다
2. **ID 토큰**: 사용자 인증 정보가 담긴 표준화된 토큰을 받을 수 있습니다
3. **더 나은 호환성**: 표준 프로토콜을 따르므로 더 안정적입니다

## 권장 사항

### ✅ 활성화 권장
- 이메일 정보가 필요한 경우
- 프로필 정보(이름, 사진 등)를 활용하고 싶은 경우
- 더 안정적인 사용자 정보 수집을 원하는 경우

### ❌ 활성화 불필요
- 단순히 로그인만 하면 되는 경우
- 최소한의 정보만 필요한 경우

## 설정 방법

1. 카카오 개발자 센터 → "제품 설정" → "카카오 로그인"
2. "OpenID Connect" 섹션 찾기
3. 활성화 (체크박스 선택)
4. 저장

## 주의사항

- OpenID Connect를 활성화해도 Supabase 설정은 동일합니다
- REST API 키와 Client Secret은 그대로 사용하면 됩니다
- 추가 설정이나 코드 변경은 필요 없습니다

## 결론

**권장**: OpenID Connect를 활성화하는 것을 권장합니다. 추가 비용이나 복잡성 없이 더 많은 사용자 정보를 안정적으로 받을 수 있습니다.
