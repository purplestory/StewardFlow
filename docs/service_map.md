# 서비스 맵

본 문서는 서비스 전반의 화면/기능 영역을 한눈에 파악하기 위한 지도입니다.

## 1) 주요 사용자 여정
- 로그인/프로필
- 조직 설정 및 권한 관리
- 자산/공간 등록 및 목록
- 예약 신청/승인/반납
- 알림/감사 로그 확인

## 2) 화면 구조 맵 (URL 기준)

### 공통
- `/` : 자산 목록
- `/login` : 로그인/프로필
- `/notifications` : 알림 리스트

### 자산
- `/assets/new` : 자산 등록
- `/assets/[id]` : 자산 상세 + 예약 신청
- `/assets/manage` : 자산 예약 관리 + 자산 관리
- `/assets/transfers` : 불용품 이동 요청

### 공간
- `/spaces` : 공간 목록
- `/spaces/new` : 공간 등록
- `/spaces/[id]` : 공간 상세 + 예약 신청
- `/spaces/manage` : 공간 예약 관리 + 공간 관리

### 설정
- `/settings/org` : 조직 설정(생성/참여)
- `/settings/approvals` : 승인 정책
- `/settings/users` : 사용자/권한/초대 관리
- `/settings/audit` : 감사 로그

## 3) 핵심 기능 맵

### 자산 관리
- 등록: 사진 업로드, 소유 범위/부서, 구입일/구입금액/사용수명
- 태그: 절기/행사/용도 등 사용자 입력 또는 AI 추천 태그
- 미사용 확인: 마지막 사용일 기반 필터로 불용품 확인
- 불용품 이동 요청: 타 부서 불용품을 우리 부서로 이동 요청
- 예약: 날짜 선택, 승인 필요 권한 표시
- 상태 관리: 일괄 변경
- 사용 이력: 예약 승인 시 마지막 사용일 업데이트

### 공간 관리
- 등록: 사진(선택), 소유 범위/부서
- 예약: 날짜 선택, 승인 필요 권한 표시
- 상태 관리: 일괄 변경

### 승인 정책
- 기관 공용/부서별 승인 권한 설정
- 템플릿 생성(기관/부서)
- 미리보기(대상/범위/부서별 적용 권한)

### 알림/감사 로그
- 예약 상태 변경 알림
- 권한/초대/승인 정책/상태 변경 기록

## 4) 데이터 테이블 맵 (핵심)
- `profiles` : 사용자/부서/권한
- `organizations` : 조직
- `assets` : 자산(감가상각 기초 필드 포함)
- `reservations` : 자산 예약
- `spaces` : 공간
- `space_reservations` : 공간 예약
- `approval_policies` : 승인 정책
- `notifications` : 알림
- `audit_logs` : 감사 로그
- `organization_invites` : 조직 초대
