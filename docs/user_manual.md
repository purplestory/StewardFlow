## 시작하기

### 1. 로그인

**카카오톡 로그인:**
1. 홈페이지에서 **"카카오톡으로 로그인"** 버튼 클릭
2. 카카오톡 계정으로 로그인
3. 권한 승인

**이메일 초대링크:**
1. 관리자로부터 초대 이메일 수신
2. 이메일 내 초대 링크 클릭
3. 가입 완료

### 2. 프로필 설정

**마이페이지 접속:**
- 상단 메뉴 > **"마이페이지"** 클릭

**프로필 수정:**
1. **"프로필 편집"** 버튼 클릭
2. 이름, 부서, 전화번호 입력/수정
3. **"저장"** 버튼 클릭

## 자원 조회

### 1. 물품 조회

**경로**: 상단 메뉴 > **"물품"**

**목록 보기:**
- 모든 물품 목록 확인
- 카드 형태로 표시
- 이미지, 이름, 상태, 소유 부서 표시

**검색:**
- 상단 검색창에서 물품명으로 검색
- 실시간 검색 결과 표시

**필터링:**
- 상태별 필터: 사용 가능 / 사용 중 / 수리 중
- 부서별 필터: 특정 부서 소유 물품만 표시

**상세 보기:**
1. 물품 카드 클릭
2. 상세 페이지에서 다음 정보 확인:
   - 물품 이미지 (여러 장)
   - 이름, 카테고리, 상태
   - 소유 부서, 관리 부서
   - 위치, 메모
   - 예약 일정 (캘린더)

### 2. 공간 조회

**경로**: 상단 메뉴 > **"공간"**

**목록 보기:**
- 물품과 동일한 방식

**상세 보기:**
- 물품과 동일한 방식
- 공간 특화 정보: 수용 인원, 시설 등

### 3. 차량 조회

**경로**: 상단 메뉴 > **"차량"**

**목록 보기:**
- 물품과 동일한 방식

**상세 보기:**
- 차량 특화 정보:
  - 차량 번호판
  - 차종 (승용차/승합차/SUV)
  - 연료 타입
  - 수용 인원
  - 현재 주행거리
  - 주차 장소

## 예약하기

### 1. 예약 신청

**물품/공간 예약:**
1. 자원 상세 페이지로 이동
2. **"예약 신청"** 버튼 클릭
3. 예약 정보 입력:
   - **시작일**: 사용 시작 날짜
   - **시작 시간**: 사용 시작 시간
   - **종료일**: 사용 종료 날짜
   - **종료 시간**: 사용 종료 시간
   - **반복 예약** (선택사항):
     - 반복 유형: 매일 / 매주 / 매월
     - 반복 간격: 1주, 2주 등
     - 반복 요일 (매주인 경우)
     - 반복 종료일
   - **메모**: 사용 목적, 특이사항 등
4. **"예약 신청"** 버튼 클릭

**차량 예약:**
1. 차량 상세 페이지로 이동
2. **"예약 신청"** 버튼 클릭
3. 예약 정보 입력:
   - 물품/공간과 동일한 항목
   - **대여 시 주행거리**: 계기판에 표시된 현재 주행거리 입력
4. **"예약 신청"** 버튼 클릭

### 2. 예약 상태

**예약 상태 종류:**
- **대기 중**: 승인 대기 중
- **승인됨**: 예약 승인 완료, 사용 가능
- **거부됨**: 예약 거부됨
- **반납됨**: 사용 완료, 반납 대기 중
- **완료됨**: 반납 및 검증 완료

**상태 확인:**
- 마이페이지 > **"내 예약"**에서 확인
- 알림으로 상태 변경 알림 수신

### 3. 반복 예약

**반복 예약 설정:**
- **매일**: 매일 반복
- **매주**: 특정 요일마다 반복
- **매월**: 매월 같은 날짜 반복

**반복 간격:**
- 1주, 2주 등 간격 설정 가능

**반복 종료일:**
- 반복 예약의 마지막 날짜 설정

**주의사항:**
- 반복 예약은 각 인스턴스별로 개별 승인됩니다
- 일부 날짜만 승인/거부될 수 있습니다

## 내 예약 관리

**경로**: 상단 메뉴 > **"마이페이지" > "내 예약"**

### 1. 예약 목록

**예약 목록 확인:**
- 모든 예약 내역 표시
- 상태별로 색상 구분:
  - 대기 중: 노란색
  - 승인됨: 초록색
  - 거부됨: 빨간색
  - 반납됨: 파란색
  - 완료됨: 회색

**필터링:**
- 상태별 필터
- 날짜별 필터

### 2. 예약 상세

**예약 상세 정보:**
- 자원 정보 (물품/공간/차량명)
- 예약 기간 (시작일시 ~ 종료일시)
- 상태
- 메모
- 승인/거부 사유 (있는 경우)

### 3. 예약 취소

**예약 취소:**
- 현재 버전에서는 예약 취소 기능이 없습니다
- 필요시 관리자에게 문의

## 반납하기

### 1. 반납 절차

**일반 물품/공간 반납:**
1. 마이페이지 > **"내 예약"**에서 **"승인됨"** 상태의 예약 찾기
2. **"반납하기"** 버튼 클릭
3. 반납 정보 입력:
   - **반납 사진**: 물품/공간 상태를 확인할 수 있는 사진 업로드 (필수)
   - **메모**: 반납 시 특이사항 (선택사항)
4. **"반납 완료"** 버튼 클릭

**차량 반납:**
1. 마이페이지 > **"내 예약"**에서 **"승인됨"** 상태의 예약 찾기
2. **"반납하기"** 버튼 클릭
3. 반납 정보 입력:
   - **계기판 사진**: 최종 주행거리가 보이는 계기판 사진 (필수)
   - **외관 사진**: 차량 외관 상태 사진 (필수)
   - **주행거리**: 계기판에 표시된 최종 주행거리 입력 (필수)
   - **메모**: 반납 시 특이사항 (선택사항)
4. **"반납 완료"** 버튼 클릭

**반납 정보 확인:**
- 대여 시 주행거리 자동 표시
- 실제 운행거리 자동 계산 및 표시

### 2. 반납 확인 대기

**반납 확인 정책에 따라:**
- **검증 불필요**: 반납 즉시 완료
- **검증 필수**: 관리자 검증 대기

**반납 상태:**
- **반납됨**: 반납 완료, 검증 대기 중
- **완료됨**: 검증 완료

### 3. 반납 거부

**반납 거부 시:**
- 관리자가 반납을 거부할 수 있습니다
- 거부 사유 확인 가능
- 필요시 재반납 가능

## 알림 확인

**경로**: 상단 메뉴 > **"알림"** (벨 아이콘)

### 1. 알림 유형

**예약 관련:**
- 예약 승인 요청 알림 (관리자용)
- 예약 승인 완료 알림
- 예약 거부 알림

**반납 관련:**
- 반납 요청 알림 (관리자용)
- 반납 검증 완료 알림
- 반납 거부 알림

### 2. 알림 확인

**알림 목록:**
- 최신 알림부터 표시
- 읽음/안 읽음 구분
- 알림 클릭 시 관련 페이지로 이동

**알림 읽음 처리:**
- 알림 클릭 시 자동 읽음 처리
- 읽지 않은 알림은 배지로 표시

## 자주 묻는 질문

### Q1. 예약은 언제까지 신청할 수 있나요?

A. 자원의 상태가 **"사용 가능"**이면 언제든지 예약 신청할 수 있습니다. 다만, 다른 사용자가 이미 예약한 기간과 겹치면 예약이 불가능할 수 있습니다.

### Q2. 예약 승인은 얼마나 걸리나요?

A. 승인 정책에 따라 다릅니다:
- **일반 사용자 승인 가능**: 즉시 승인
- **부서 관리자 승인 필요**: 부서 관리자가 확인 후 승인
- **관리자 승인 필요**: 관리자가 확인 후 승인

### Q3. 반복 예약은 어떻게 되나요?

A. 반복 예약은 각 인스턴스별로 개별 승인됩니다. 예를 들어, 매주 월요일 예약을 신청하면 각 주의 월요일 예약이 개별적으로 승인/거부됩니다.

### Q4. 차량 반납 시 주행거리는 어떻게 입력하나요?

A. 차량 반납 시:
1. 계기판에 표시된 최종 주행거리를 확인
2. 계기판 사진 촬영
3. 입력 필드에 주행거리 숫자 입력
4. 실제 운행거리는 자동으로 계산되어 표시됩니다

### Q5. 예약을 취소하고 싶어요.

A. 현재 버전에서는 사용자가 직접 예약을 취소할 수 없습니다. 예약 취소가 필요하면 관리자에게 문의하세요.

### Q6. 반납 사진은 필수인가요?

A. 기관 설정에서 **"반납 확인 정책"**이 활성화되어 있고 **"사진 필수"**가 설정되어 있으면 반납 사진은 필수입니다. 정책에 따라 다를 수 있으니 확인해주세요.

### Q7. 알림이 오지 않아요.

A. 다음을 확인해주세요:
- 알림 설정이 활성화되어 있는지 확인
- 브라우저 알림 권한 확인
- 알림 페이지에서 알림 목록 확인

### Q8. 물품/공간/차량을 등록하고 싶어요.

A. 자원 등록은 **관리자** 또는 **부서 관리자**만 가능합니다. 등록이 필요하면 관리자에게 문의하세요.

### Q9. 내 부서 물품만 보고 싶어요.

A. 물품 목록 페이지에서 **"부서 필터"**를 사용하여 본인 부서 물품만 필터링할 수 있습니다.

### Q10. 예약이 거부되었어요. 이유를 알고 싶어요.

A. 마이페이지 > **"내 예약"**에서 거부된 예약을 클릭하면 거부 사유를 확인할 수 있습니다.

## 주요 기능 요약

### ✅ 할 수 있는 것
- 자원 조회 및 검색
- 예약 신청 (일회성 및 반복)
- 내 예약 확인
- 반납 처리
- 알림 확인
- 프로필 수정

### ❌ 할 수 없는 것
- 자원 등록/수정/삭제 (관리자만 가능)
- 예약 승인/거부 (관리자만 가능)
- 반납 검증 (관리자만 가능)
- 기관 설정 변경 (관리자만 가능)
