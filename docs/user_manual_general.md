## 시작하기

### 1. 로그인

**카카오톡 로그인:**
1. 홈페이지에서 **"카카오톡으로 로그인"** 버튼 클릭
2. 카카오톡 계정으로 로그인
3. 권한 승인

**초대 링크로 가입:**
1. 관리자로부터 초대 링크 수신 (이메일 또는 직접 전달)
2. 초대 링크 클릭
3. 가입 페이지에서 정보 확인 및 수정
4. 가입 완료

### 2. 첫 화면 이해하기

**상단 메뉴:**
- **물품**: 교회 내 물품 목록
- **공간**: 교회 내 공간 목록
- **차량**: 교회 내 차량 목록
- **마이페이지**: 내 예약, 프로필 관리
- **알림**: 알림 확인 (벨 아이콘)

**사용자 메뉴 (우측 상단):**
- 사용자 이름 클릭 시 드롭다운 메뉴 표시
- 마이페이지, 관리페이지 (관리자/부서 관리자만), 로그아웃

## 자원 조회 및 검색

### 1. 물품 조회

**경로**: 상단 메뉴 > **"물품"**

**목록 보기:**
- 모든 물품이 카드 형태로 표시됩니다
- 각 카드에는 다음 정보가 표시됩니다:
  - 물품 이미지
  - 물품명
  - 상태 뱃지 (대여 중, 수리 중, 분실, 불용품)
  - 소유 부서
  - 설치 형태 (이동/고정)
  - 태그 (카테고리 등)
  - 승인 필요 표시 (있는 경우)

**검색:**
- 상단 검색창에서 물품명, 부서명, 태그로 검색
- 실시간으로 검색 결과가 필터링됩니다

**카테고리 필터:**
- 상단 카테고리 선택 드롭다운에서 카테고리 선택
- 선택한 카테고리만 표시됩니다

**상태 필터:**
- 상단 필터 버튼으로 상태별 필터링:
  - **전체**: 모든 물품
  - **대여 가능**: 대여 가능한 물품만
  - **대여 중**: 현재 대여 중인 물품
  - **수리 중**: 수리 중인 물품
  - **분실**: 분실된 물품
  - **불용품**: 불용품으로 지정된 물품

**상세 보기:**
1. 물품 카드 클릭
2. 상세 페이지에서 다음 정보 확인:
   - 물품 이미지 슬라이더 (여러 장)
   - 물품명, 카테고리
   - 상태 뱃지
   - 소유 부서, 설치 형태
   - 설치(보관) 장소
   - 구입 정보 (구입일, 구입금액, 사용 수명, 사용 기한)
   - 태그
   - 예약 일정 캘린더
   - 예약 신청 버튼

### 2. 공간 조회

**경로**: 상단 메뉴 > **"공간"**

**목록 보기:**
- 물품과 동일한 방식으로 표시됩니다

**상세 보기:**
- 물품과 동일한 정보 구조
- 추가 정보:
  - 수용 인원
  - 비고 (사용 시 주의사항 등)

### 3. 차량 조회

**경로**: 상단 메뉴 > **"차량"**

**목록 보기:**
- 물품과 동일한 방식으로 표시됩니다

**상세 보기:**
- 차량 특화 정보:
  - 차량 번호판
  - 차종 (승용차/승합차/SUV 등)
  - 연료 타입 (가솔린/디젤/전기 등)
  - 수용 인원
  - 현재 주행거리
  - 주차 장소
  - 비고

## 예약 신청

### 1. 예약 신청 절차

**물품/공간 예약:**
1. 자원 상세 페이지로 이동
2. **"예약 신청"** 버튼 클릭
3. 예약 정보 입력:
   - **시작일시**: 사용 시작 날짜와 시간
   - **종료일시**: 사용 종료 날짜와 시간
   - **반복 유형** (선택사항):
     - 없음: 일회성 예약
     - 매주: 매주 반복
     - 매월: 매월 반복
   - **반복 설정** (반복 유형 선택 시):
     - 반복 간격: 1주, 2주 등
     - 반복 종료일: 반복 예약의 마지막 날짜
     - 반복 요일 (매주인 경우): 요일 선택
     - 반복 일 (매월인 경우): 날짜 선택
   - **사용 목적**: 사용 목적 및 특이사항 입력
4. **"예약 신청"** 버튼 클릭

**차량 예약:**
1. 차량 상세 페이지로 이동
2. **"예약 신청"** 버튼 클릭
3. 예약 정보 입력:
   - 물품/공간과 동일한 항목
   - **대여 시 초기 주행거리**: 계기판에 표시된 현재 주행거리 입력 (필수)
4. **"예약 신청"** 버튼 클릭

### 2. 예약 상태 이해하기

**예약 상태 종류:**
- **승인 대기**: 관리자 승인을 기다리는 중
- **승인됨**: 예약이 승인되어 사용 가능
- **반납 완료**: 사용 완료 후 반납 처리됨
- **반려**: 예약이 거부됨

**상태 확인 방법:**
- 마이페이지 > **"내 대여 신청"**에서 확인
- 알림으로 상태 변경 알림 수신
- 각 예약 카드에 상태 뱃지 표시

### 3. 반복 예약 이해하기

**반복 예약이란?**
- 같은 자원을 정기적으로 사용할 때 반복 예약을 설정할 수 있습니다
- 예: 매주 월요일 오후 2시에 찬양팀 연습실 사용

**반복 예약의 특징:**
- 각 반복 인스턴스는 개별적으로 승인됩니다
- 일부 날짜만 승인/거부될 수 있습니다
- 반복 종료일까지 자동으로 예약이 생성됩니다

**반복 예약 예시:**
- **매주 반복**: 매주 월요일, 2주 간격, 2025년 3월까지
- **매월 반복**: 매월 1일, 1개월 간격, 2025년 12월까지

## 내 예약 관리

**경로**: 상단 메뉴 > **"마이페이지" > "내 대여 신청"**

### 1. 예약 목록 확인

**예약 목록:**
- 모든 예약 내역이 카드 형태로 표시됩니다
- 각 카드에는 다음 정보가 표시됩니다:
  - 자원 이미지
  - 자원명 (물품/공간/차량)
  - 예약 기간 (시작일시 ~ 종료일시)
  - 상태 뱃지
  - 반납 버튼 (승인됨 상태일 때만)

**상태별 색상:**
- **승인 대기**: 노란색 뱃지
- **승인됨**: 초록색 뱃지
- **반납 완료**: 파란색 뱃지
- **반려**: 빨간색 뱃지

**예약 상세:**
- 예약 카드 클릭 시 상세 정보 확인:
  - 자원 정보
  - 예약 기간
  - 상태
  - 사용 목적
  - 반려 사유 (반려된 경우)

### 2. 예약 취소

**현재 버전:**
- 사용자가 직접 예약을 취소할 수 없습니다
- 예약 취소가 필요하면 관리자에게 문의하세요

## 반납하기

### 1. 반납 절차

**일반 물품/공간 반납:**
1. 마이페이지 > **"내 대여 신청"**에서 **"승인됨"** 상태의 예약 찾기
2. **"반납하기"** 버튼 클릭
3. 반납 정보 입력:
   - **반납 사진**: 물품/공간 상태를 확인할 수 있는 사진 업로드
     - 정책에 따라 필수일 수 있습니다
     - 여러 장 업로드 가능
   - **메모**: 반납 시 특이사항 (선택사항)
4. **"반납 완료"** 버튼 클릭

**차량 반납:**
1. 마이페이지 > **"내 대여 신청"**에서 **"승인됨"** 상태의 예약 찾기
2. **"반납하기"** 버튼 클릭
3. 반납 정보 입력:
   - **계기판 사진**: 최종 주행거리가 보이는 계기판 사진 (필수)
   - **외관 사진**: 차량 외관 상태 사진 (필수)
   - **반납 시 주행거리**: 계기판에 표시된 최종 주행거리 입력 (필수)
   - **메모**: 반납 시 특이사항 (선택사항)
4. **"반납 완료"** 버튼 클릭

**반납 정보 확인:**
- 대여 시 주행거리 자동 표시 (차량의 경우)
- 실제 운행거리 자동 계산 및 표시 (차량의 경우)

### 2. 반납 확인 대기

**반납 확인 정책:**
- 기관 설정에 따라 반납 확인 절차가 다릅니다:
  - **검증 불필요**: 반납 즉시 완료
  - **검증 필수**: 관리자 검증 대기

**반납 상태:**
- **반납 완료**: 반납 처리 완료, 검증 대기 중 (검증 필수인 경우)
- **완료됨**: 검증 완료 또는 즉시 완료

### 3. 반납 거부

**반납 거부 시:**
- 관리자가 반납을 거부할 수 있습니다
- 거부 사유 확인 가능
- 필요시 재반납 가능

## 알림 확인

**경로**: 상단 메뉴 > **"알림"** (벨 아이콘) 또는 상단 우측 알림 아이콘

### 1. 알림 유형

**예약 관련:**
- 예약 승인 완료 알림
- 예약 반려 알림

**반납 관련:**
- 반납 검증 완료 알림
- 반납 거부 알림

### 2. 알림 확인

**알림 목록:**
- 최신 알림부터 표시
- 읽음/안 읽음 구분
- 알림 클릭 시 관련 페이지로 이동

**알림 읽음 처리:**
- 알림 클릭 시 자동 읽음 처리
- 읽지 않은 알림은 배지로 표시
- **"모두 읽음으로 표시"** 버튼으로 일괄 처리 가능

**알림 필터:**
- 전체 알림
- 읽지 않은 알림만
- 읽은 알림만

**표시되는 정보:**

## 자주 묻는 질문

### Q1. 예약은 언제까지 신청할 수 있나요?

A. 자원의 상태가 **"대여 가능"**이고 다른 예약과 겹치지 않으면 언제든지 예약 신청할 수 있습니다. 다만, 고정 설치된 자산이나 대여 불가로 설정된 자산은 예약할 수 없습니다.

### Q2. 예약 승인은 얼마나 걸리나요?

A. 승인 정책에 따라 다릅니다:
- **일반 사용자 승인 가능**: 즉시 승인
- **부서 관리자 승인 필요**: 부서 관리자가 확인 후 승인
- **관리자 승인 필요**: 관리자가 확인 후 승인

승인 정책은 기관 설정에서 관리자가 설정합니다.

### Q3. 반복 예약은 어떻게 되나요?

A. 반복 예약은 각 인스턴스별로 개별 승인됩니다. 예를 들어, 매주 월요일 예약을 신청하면 각 주의 월요일 예약이 개별적으로 승인/거부됩니다.

### Q4. 차량 반납 시 주행거리는 어떻게 입력하나요?

A. 차량 반납 시:
1. 계기판에 표시된 최종 주행거리를 확인
2. 계기판 사진 촬영
3. 입력 필드에 주행거리 숫자 입력
4. 실제 운행거리는 자동으로 계산되어 표시됩니다

### Q5. 예약을 취소하고 싶어요.

A. 현재 버전에서는 사용자가 직접 예약을 취소할 수 없습니다. 예약 취소가 필요하면 관리자에게 문의하세요.

### Q6. 반납 사진은 필수인가요?

A. 기관 설정에서 **"반납 확인 정책"**이 활성화되어 있고 **"사진 필수"**가 설정되어 있으면 반납 사진은 필수입니다. 정책에 따라 다를 수 있으니 확인해주세요.

### Q7. 알림이 오지 않아요.

A. 다음을 확인해주세요:
- 알림 페이지에서 알림 목록 확인
- 브라우저 알림 권한 확인
- 알림이 이미 읽음 처리되었는지 확인

### Q8. 물품/공간/차량을 등록하고 싶어요.

A. 자원 등록은 **관리자** 또는 **부서 관리자**만 가능합니다. 등록이 필요하면 관리자에게 문의하세요.

### Q9. 내 부서 물품만 보고 싶어요.

A. 물품 목록 페이지에서 검색창에 부서명을 입력하거나, 부서 필터를 사용하여 본인 부서 물품만 필터링할 수 있습니다.

### Q10. 예약이 거부되었어요. 이유를 알고 싶어요.

A. 마이페이지 > **"내 대여 신청"**에서 거부된 예약을 클릭하면 거부 사유를 확인할 수 있습니다.

### Q11. "대여 불가"라고 표시되는 이유는?

A. 다음 경우에 "대여 불가"로 표시됩니다:
- 자원이 고정 설치되어 있는 경우
- 관리자가 대여 불가로 설정한 경우
- 현재 대여 중이거나 수리 중인 경우
